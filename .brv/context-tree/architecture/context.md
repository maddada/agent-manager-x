# Domain: architecture

## Purpose
Contains architecture and integration knowledge for how Agent Manager X discovers, models, and interacts with agent sessions.

## Scope
Included in this domain:
- Session retrieval modes and lifecycle
- Session brokers, polling, and push-based updates
- Session models and UI interaction flows
- Cross-component session focus and open behavior

Excluded from this domain:
- General release packaging details
- Unrelated UI styling outside session workflows

## Ownership
Agent Manager X application codebase

## Usage
Use this domain for documenting how sessions are sourced, transformed, rendered, and acted on across the app.
